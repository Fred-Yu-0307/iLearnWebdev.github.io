Frontend Development

Introduction

Front-end development refers to the part of web development that deals with the user interface (UI) and user experience (UX) of a website or web application. Front-end developers are responsible for creating the visual design and user interface of a website or application using a combination of programming languages, markup languages, and style sheet languages.

Front-end development typically involves HTML, CSS, and JavaScript, as well as various frameworks and libraries that facilitate the development process. HTML provides the structure of a website, CSS is used to style and layout the content, and JavaScript is used to add interactivity and functionality.

Authentication Strategies


Authentication strategies are methods or techniques used to verify the identity of a user or system in order to grant access to a protected resource. There are several different authentication strategies that can be used, including:

• Basic Authentication
• Session Based Authentication
• Token Based Authentication
• JWT Authentication
• OAuth
• SSO

You don’t necessarily need to learn all of these, how to implement and the ins and outs from the get-go. But it’s important to know what they are and how they work. This will help you make better decisions when choosing an authentication strategy for your application.

  • Basic Authentication
